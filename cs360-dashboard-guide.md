# CS360 Interactive Dashboard - Complete HTML Setup

This is a comprehensive guide to create an interactive HTML dashboard for CareStream360 (cs360) with multiple linked pages showcasing QoE metrics, customer data, and real-time monitoring.

## Files to Create:

1. `index.html` - Main dashboard home page
2. `pages/sessions.html` - Active sessions monitor
3. `pages/qoe-analytics.html` - QoE metrics and trends
4. `pages/customers.html` - Customer management
5. `pages/alerts.html` - System alerts and notifications
6. `css/style.css` - Global styling
7. `js/script.js` - Shared JavaScript functionality
8. `js/charts.js` - Chart library integration
